How to use these

  1. Open this file in any browser to preview the four templates.
  2. Right-click each pin → Save image as… to grab the rendered design.
  3. Recreate in Canva with the listed specs, swapping in real product photos from site/public/products/.
  4. Export at 1000×1500 px (Pinterest 2:3 ratio).
  5. Make 3-5 variations of each template per niche so pins look fresh, not bot-repeated.
